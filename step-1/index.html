Step 1

Add a Calcite Shell component to contain our application and map container 

<calcite-shell>
    <div id="viewDiv"></div>
</calcite-shell>
Calcite Shell Calcite Shell is a foundational layout component that contains your application or experience. It can be full-screen or embedded in narrative content like a blog article.

Add a Calcite Navigation and Calcite Navigation Logo components. 

<calcite-shell>
    <calcite-navigation>
        <calcite-navigation-logo></calcite-navigation-logo>
        <calcite-menu slot="content-end"></calcite-menu>
    </calcite-navigation>
    <div id="viewDiv"></div>
</calcite-shell>

Place a Map and MapView in the default slot of Calcite Shell 

const map = new Map({
    basemap: "streets-night-vector",
});

const view = new MapView({
    container: "viewDiv",
    map,
    center: [-100, 45],
    zoom: 3,
});
