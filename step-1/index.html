About this application
Step 1: Construct our Application layout

Add a Calcite Shell component to contain our application and map container 

<calcite-shell>
    <div id="viewDiv"></div>
</calcite-shell>

Add a Calcite Navigation, Calcite Navigation Logo, and Calcite Navigation User components. 

<calcite-shell>

    <calcite-navigation slot="header">
        <calcite-navigation-logo slot="logo" heading="Alternative Fuel Stations" thumbnail="./logo.svg"></calcite-navigation-logo>
        <calcite-navigation-user slot="user" full-name="Sample user" username="map_fan_123" ></calcite-navigation-user>
    </calcite-navigation>
    
    <div id="viewDiv"></div>
</calcite-shell>

Place a Map and MapView in the default slot of Calcite Shell 

const map = new Map({
    basemap: "streets-night-vector",
});

const view = new MapView({
    container: "viewDiv",
    map,
    center: [-100, 45],
    zoom: 3,
});
